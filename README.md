## 画像のダミーデータ
public/imagesフォルダ内にsample1.jpeg～sample6.jpegとして保存しています。  
```php artisan storage:link```  
コマンドの実行後、storageフォルダにリンク後、  
storage/app/public/productsフォルダ内に保存すると表示されます。  
productsフォルダがない場合は作成してください。
